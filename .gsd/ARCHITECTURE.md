## Overview
The **Pet Membership Form** is a full-stack Next.js application designed to manage pet registrations and memberships for "Club Pata Amiga". It integrates with Supabase for data persistence, Memberstack for user authentication and membership management, Sanity for CMS/content, and Stripe/Resend for payments and notifications.

### Services (`src/services`)
- **Supabase**: Data access layer for pets, owners, and clinical records.
- **Memberstack**: Membership state and user profile synchronization.
- **Resend**: Transactional email services.

## Data Flow
1. **User Landing**: User visits the registration form.
2. **Step-by-Step Registration**: Data is collected through multiple steps (Owner → Pet → Clinical History → Membership).
3. **Draft Saving**: Intermediate steps are saved to Supabase to prevent data loss.
4. **Completion**: On final submission, Memberstack profile is updated, and a welcome email is sent via Resend.
