# Changelog - 2026-05-27

## feat: refactor complete-profile-widget with dynamic missing fields and full pet registration flow
**Commit:** `e57706d`

### Cambios Principales

#### `public/widgets/complete-profile-widget.js`

**1. `renderCompletePetForm()` — Solo muestra campos faltantes**
- Nueva función `getMissingFields(pet)` que analiza cada mascota y retorna un array con los campos específicos que faltan (`gender`, `breed`, `breedType`, `coatColor`, `noseColor`, `eyeColor`, `photo`, `vetCert`)
- El formulario ahora **solo renderiza los inputs de los campos que realmente le faltan** a la mascota, en vez del formulario completo
- Muestra un info box informativo: *"A **Luna** le faltan X datos por completar"*
- Si un usuario ya tiene todas sus mascotas completas → no le pide nada, va directo a "perfil completo"

**2. `renderAddPetForm()` — Mezcla real de Step 2 + Step 5**
- Organizado con secciones claras: 🐾 Datos básicos → 📋 Información general → 🎨 Características físicas → 🏠 Adopción → 📷 Fotografía
- **Breed type switch** con botones toggle (Mestizo/Raza) idéntico al flujo real del Step 5 del registro
- **Breed autocomplete** con búsqueda contra `/api/breeds` (debounced, 300ms)
- El label del botón Mestizo cambia dinámicamente a "Doméstico" al seleccionar Gato
- Campos de color (pelo, nariz, ojos) incluidos
- Sección de adopción con historia y aviso de autorización de publicación

**3. `determineSteps()` — Lógica mejorada**
- Almacena `this.incompletePetMissingFields` para que `renderCompletePetForm` sepa exactamente qué mostrar
- La validación de submit solo verifica los campos que realmente estaban faltando

**4. Nuevos estilos CSS**
- `.ppa-breed-switch` / `.ppa-breed-switch-btn` — Toggle buttons para Mestizo/Raza
- `.ppa-section-title` — Títulos de sección con font Fraiche
- `.ppa-adoption-box` / `.ppa-adoption-notice` — Sección de adopción
- `.ppa-autocomplete-wrapper` / `.ppa-autocomplete-list` / `.ppa-autocomplete-item` — Autocomplete de razas
- `.ppa-info-box` — Info box para mensajes contextuales
- `.ppa-btn-secondary` — Botón secundario turquesa

**5. Nueva función `setupBreedAutocomplete()`**
- Reutilizable entre `renderAddPetForm` y `renderCompletePetForm`
- Busca razas contra `/api/breeds?type=X&search=Y`
- Debounce de 300ms, muestra hasta 8 resultados
- Click fuera cierra el dropdown

### Impacto Funcional
- Usuarios con mascotas completas ya no ven formularios innecesarios
- Usuarios con mascotas incompletas solo ven los campos que les faltan
- El registro de primera mascota desde el widget ahora es consistente con el flujo real de la app
