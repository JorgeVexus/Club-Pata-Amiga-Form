# Changelog — 2026-04-28

## feat: Visualización de Fechas de Facturación en Detalle de Miembro

### Resumen
Se enriqueció el modal de detalles de miembros para proporcionar transparencia total sobre el ciclo de facturación de las suscripciones. Los administradores ahora pueden ver la fecha exacta en que se contrató la membresía y cuándo es la próxima renovación, facilitando el soporte técnico y la auditoría de cuentas.

### Archivos Modificados

#### `src/components/Admin/MemberDetailModal.tsx`
- **Nueva Sección de Membresía**: Implementación de un bloque dedicado "Detalles de Membresía 💳" con diseño diferenciado (borde turquesa).
- **Extracción de Datos**: Lógica para parsear `createdAt` y `currentPeriodEnd` desde el objeto `planConnections` de Memberstack.
- **Soporte de Formatos**: Manejo de timestamps de Unix (segundos) y strings ISO para asegurar la compatibilidad con diferentes versiones de la API de Memberstack.
- **Localización**: Uso de `toLocaleDateString` con configuración `es-MX` para mostrar nombres de meses en español y formato de fecha local.

#### `src/components/Admin/MemberDetailModal.module.css`
- **Estilos de Sección**: Adición de clases para la sección de facturación y mensajes de "Sin Plan".
- **Badges de Pago**: Integración de estilos para badges de estado (Active, Past Due, Canceled, etc.) dentro del modal para mantener la consistencia con la tabla principal.

---

### Impacto funcional
- **Claridad Operativa**: Los administradores pueden responder dudas sobre renovaciones sin tener que consultar el dashboard externo de Memberstack.
- **Auditoría Rápida**: Identificación inmediata de cuándo comenzó la relación del miembro con el club y su estado de suscripción actual.
- **Estética Coherente**: Se mantiene el lenguaje visual de "badges" y "grids" establecido en el resto del dashboard administrativo.
