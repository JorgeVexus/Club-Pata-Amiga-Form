# Changelog — 2026-04-24

## feat: Habilitación de carga de certificados PDF para mascotas

### Resumen
Se habilitó el soporte para archivos PDF en los flujos de certificación veterinaria de la plataforma, permitiendo a los usuarios subir documentos médicos en un formato más común además de imágenes tradicionales.

---

### Archivos Modificados

#### `src/app/api/upload/pet-photo/route.ts`
- Se actualizó `MAX_FILE_SIZE` de 5MB a **10MB**.
- Se añadió `application/pdf` a la lista de tipos permitidos (`ALLOWED_MIME_TYPES`).
- Se actualizó el mensaje de error para reflejar el soporte de PDF y el nuevo límite de tamaño.

#### `src/app/api/user/upload-pet-photo/route.ts`
- Se sincronizó la validación del lado del servidor para permitir archivos `application/pdf`.
- Se reforzó el límite de tamaño a 10MB para mantener consistencia con la API de carga general.

#### `public/widgets/unified-membership-widget.js`
- Se actualizó la validación del lado del cliente en `uploadPhoto` para permitir archivos de hasta **10MB**.
- Se validó la lógica de previsualización para mostrar un icono de documento (📄) cuando se carga un archivo no-imagen (PDF), evitando errores de renderizado.

#### `public/widgets/pet-cards-widget.js`
- Se añadió validación explícita del lado del cliente en `uploadNewPetPhoto` para tipos de archivo (JPG, PNG, PDF) y tamaño (10MB).
- Se actualizó el límite de tamaño a 10MB.
- Se verificó que los modales de carga manejen correctamente el estado visual durante el procesamiento de PDFs.

---

### Impacto funcional

- **Flexibilidad**: Los usuarios ya no están limitados a tomar fotos de sus certificados; ahora pueden subir el archivo digital (.pdf) directamente desde su dispositivo.
- **Robustez**: Se previene el error 400 que ocurría al intentar subir PDFs a un endpoint que solo aceptaba imágenes.
- **Experiencia de Usuario (UX)**: Los widgets muestran retroalimentación visual adecuada para documentos PDF, manteniendo una interfaz limpia y profesional.
- **Accesibilidad**: Los certificados subidos como PDF se abren correctamente en una nueva pestaña del navegador desde el expediente de la mascota en el widget.
