# Changelog - 2026-07-03

## fix: sync wellness profile updates with admin notifications

### Cambios principales
- Se agregó una notificación administrativa cuando un Centro de Bienestar actualiza su logo desde el widget.
- El dashboard admin ahora puede abrir directamente el detalle de un centro usando `wellnessCenterId` desde la URL o desde una notificación.
- Se amplió el tipado de notificaciones del admin para aceptar datos embebidos del centro actualizado.
- Se agregó una prueba de regresión para validar la sincronización entre actualizaciones del centro y la navegación del admin.

### Impacto funcional
- El admin puede revisar más rápido los cambios de perfil de los centros sin buscar manualmente el registro.
- Los cambios capturados por el centro quedan conectados al flujo de revisión del dashboard administrativo.
- Se reduce el riesgo de que actualizaciones importantes pasen desapercibidas durante la validación de aliados.
