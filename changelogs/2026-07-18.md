# Changelog — 18 de julio de 2026

## `37de137` — `fix: make member notifications responsive and actionable`

- Se corrigió el solapamiento de las notificaciones en dispositivos móviles aislando las dimensiones de las tarjetas frente a estilos globales de Webflow.
- Se eliminó el parpadeo periódico: el preview deja de hacer polling y producción solo repinta la campana cuando cambian los datos, con consultas cada 15 segundos.
- Se añadió un resolvedor seguro de acciones que abre el expediente y chat de una mascota, el detalle interno de un reintegro o un modal informativo según el contexto.
- Se conservaron las notificaciones históricas mediante traducción de enlaces antiguos y se bloquearon destinos externos no autorizados.
- Las notificaciones nuevas de solicitudes de información, apelaciones, estados de mascotas y Fondo Solidario ahora incluyen metadata canónica.
- Se añadieron pruebas de responsive, seguridad, compatibilidad histórica, productores backend y estabilidad del polling.

### Impacto funcional

Los miembros pueden leer las notificaciones correctamente en celular y acceder directamente a la acción relacionada sin salir innecesariamente del dashboard unificado. Las actualizaciones periódicas ya no generan recargas visuales cuando no existen novedades.
