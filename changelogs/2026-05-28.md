# Changelog - 2026-05-28

## feat: show pet nose and eye color in admin member detail modal
**Commit:** `0dfedeb`

### Cambios Principales

#### `src/components/Admin/MemberDetailModal.tsx`

- Se modificaron los detalles de visualización de mascotas de miembros en el panel de administración.
- Se quitaron las condicionales `{pet.nose_color && ...}` y `{pet.eye_color && ...}` de modo que las filas de **Nariz** y **Ojos** se renderizan siempre en la cuadrícula de detalles de la mascota, al igual que los demás campos principales.
- Se implementó un mecanismo de fallback dinámico: si el campo correspondiente en Supabase (`pet.nose_color` / `pet.eye_color`) está vacío, se intenta recuperar el valor desde los custom fields correspondientes de Memberstack (`fields['pet-X-nose-color']` / `fields['pet-X-eye-color']`), y si tampoco existe, se muestra `---`.
- Se aplicó el mismo mecanismo de fallback para el color de pelo (`coat_color`).

### Impacto Funcional
- Los administradores ahora pueden ver con claridad el color de nariz y ojos de las mascotas registradas desde cualquier interfaz (Supabase o Memberstack), garantizando consistencia en el expediente de la mascota y paridad con los datos del flujo de registro.
