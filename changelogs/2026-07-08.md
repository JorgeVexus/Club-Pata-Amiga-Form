# Changelog - 2026-07-08

## feat: add welcome modal for approved ambassadors

**Commit:** `39fe969`

Se implementó un modal de bienvenida dinámico que se muestra la primera vez que un embajador aprobado entra a su dashboard.

### Cambios principales
- **Base de Datos (Supabase)**: Creado script de migración SQL `supabase/migrations/20260708_add_welcome_shown_ambassadors.sql` para añadir la columna `welcome_shown BOOLEAN DEFAULT FALSE` en la tabla `ambassadors`.
- **Estructura de API y Tipos**:
  * Agregado `welcome_shown?: boolean;` a la interfaz `Ambassador` en `src/types/ambassador.types.ts`.
  * Añadida la columna `'welcome_shown'` al listado de campos modificables (`allowedFields`) en la llamada `PATCH` a `/api/ambassadors/[id]`.
- **Widget de Webflow (`ambassador-widget.js`)**:
  * Añadidos estilos específicos para el overlay del modal y el contenedor de bienvenida con animaciones suaves de entrada.
  * Diseñado el modal con el texto de felicitación aprobado.
  * **Cierre por Click Fuera (Overlay)**: Se estructuró el cierre del modal tanto al hacer click en "Comenzar ahora" como al hacer click en el overlay oscuro. Ambas acciones de cierre envían la solicitud fetch `PATCH` para guardar `welcome_shown: true` en la base de datos de Supabase, desactivando la visualización futura.

### Impacto funcional
- Los nuevos embajadores aprobados son recibidos con un modal explicativo al ingresar por primera vez a su dashboard, facilitando su comprensión de las funcionalidades disponibles (códigos, comisiones y materiales).

### Verificación
- `npm run type-check` completado exitosamente (0 errores).
- `npm run lint` verificado sin errores.
- `npm run build` compilado exitosamente.

## feat: show profile edit modal in place for pending ambassadors

**Commit:** `029a58d`

Se modificó el flujo del botón "Completa tu perfil" en la tarjeta de revisión (estado pendiente) del widget para abrir el modal localmente.

### Cambios principales
- **Modificación de Acción del Botón**: Cambiado `onclick` del botón en `renderPending` de `public/widgets/ambassador-widget.js` para que ejecute `window.editAmbassadorProfile()` en lugar de redirigir al usuario hacia la página standalone.
- **Mensaje de Éxito**: Actualizada la respuesta en `window.submitProfileForm` para que muestre una alerta con el texto *"✅ ¡Datos actualizados! Muchas gracias..."*, cerrando el modal e inicializando el widget nuevamente con la sesión activa.

### Impacto funcional
- Los embajadores pendientes de aprobación pueden capturar sus datos de RFC, información bancaria y foto de perfil en el mismo dashboard del widget sin redireccionamientos ni flujos de login redundantes.
