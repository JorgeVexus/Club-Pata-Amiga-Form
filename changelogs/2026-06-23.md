# Changelog 2026-06-23

## ✨ Features

### Ajustes Estéticos, Alineación Móvil y Animaciones Scroll en Home Page Widget

Se han actualizado los estilos visuales, la navegación del footer, la terminología en general de mascotas a "peludos", la alineación de viñetas en pantallas móviles, y se inyectaron animaciones tipo slide-in bottom al hacer scroll.

#### Cambios implementados:

1. **Footer (Navegación limpia)**
   - Se removió el enlace de navegación "Nosotros" del footer (`INFORMACIÓN`), ya que no existe dicha sección.

2. **Hero (Mayor Ancho y Separación)**
   - Se aumentó el ancho máximo de la sección `.pata-hero-section` a `1400px` para aprovechar pantallas más amplias.
   - Se incrementó el espaciado (`gap`) a `80px` entre la columna de texto y la imagen.
   - Se incrementó el ancho máximo de `.pata-hero-content` a `680px` y se actualizó el texto a "para tu peludo", logrando que la frase quede en una sola línea en desktop.

3. **Alineación de Viñetas en Móviles**
   - Se corrigió el comportamiento en pantallas móviles donde las viñetas se alineaban individualmente al centro, creando un efecto escalonado desprolijo.
   - Ahora el bloque de viñetas está centrado como contenedor (`max-width: 290px; margin: 0 auto;`), pero cada elemento de característica está alineado limpiamente hacia la izquierda, manteniendo sus iconos verticales perfectamente alineados.

4. **Animaciones de Entrada (Slide-in)**
   - Se añadió la clase CSS `.pata-animate-on-scroll` con transiciones basadas en una curva premium (`cubic-bezier(0.16, 1, 0.3, 1)` / *easeOutExpo*) y una animación sutil hacia arriba (`translateY(30px)`).
   - Se implementó un `IntersectionObserver` de JavaScript para activar dinámicamente la visibilidad de cada sección al hacer scroll de forma progresiva y elegante, con soporte fallback para navegadores antiguos.

5. **Terminología a "Peludo/Peludos"**
   - Se reemplazaron sistemáticamente las palabras "mascota" y "mascotas" por "peludo" y "peludos" en todos los textos del widget y en los atributos `alt` de las imágenes.
   - Se actualizó el valor y la etiqueta del checkbox de "Tienda de mascotas" a "Tienda para peludos" en el formulario de registro de centros de bienestar.

6. **Redirecciones Absolutas (Corrección de Dominio)**
   - Se corrigieron los enlaces de registro y el `successUrl` del checkout de Memberstack en el widget para usar rutas absolutas (`CONFIG.apiUrl`) en lugar de relativas, evitando que la navegación redirigiera incorrectamente al dominio de Webflow.
